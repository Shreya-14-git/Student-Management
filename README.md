# Student Management System (C Programming)

## Description
A console-based Student Management System implemented in C using a singly linked list.
The project allows dynamic management of student records.

## Features
- Add student record
- Display all students
- Search student by roll number
- Update student details
- Delete student record
- Menu-driven application

## Data Structure Used
- Singly Linked List

## Technologies Used
- C Programming
- Pointers
- Dynamic Memory Allocation

## How to Run
1. Compile the program:
   gcc student_management.c -o student
2. Run the program:
   ./student
